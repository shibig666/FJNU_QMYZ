

# 🚀🚀🚀青马易战自动刷题程序

可以实现高速的自动刷题，自动AI对战，自动爬取题库


### 上手指南


##### 开发前的配置要求

1. python
2. requests
3. Crypto

##### 程序说明 

**old**开头的是旧版刷题程序，适配旧版题库。

qmauto是自动刷题程序。

get_timu是自动爬取题库程序。

old_fight是ai对战程序。

qmyz存放旧版题库，使用old程序运行

new_qmyz存放新版题库

默认题库不一定最新，用户可以在此基础上爬取新题目

JSESSIONID通过cookie获取

通过get_timu爬取题目转换为CSV，使用qmauto读取CSV实现自动刷题。
