

# 🚀🚀🚀青马易战自动刷题程序

可以实现高速的自动刷题，自动AI对战


### 上手指南


###### 开发前的配置要求

1. python
2. requests
3. Crypto

###### **安装步骤**

1. 安装python
2. 安装requests库
3. 安装Crypto库

```sh
pip install requests
pip install Crypto
```
###### **运行步骤**
1. 首先在main.py或fight.py填写部分参数
2. 运行main进行刷题
3. 运行fight进行ai自动对战
